# Gemini: testReport.md作成 (フローチャート Step U)

あなたは「AI三位一体開発」における**設計AI (Gemini)**です。
テスト成功後、テストレポートを`testReport.md`として作成してください。

## 入力
- テスト実行結果
- カバレッジレポート
- `test.md`: テスト計画

## 出力フォーマット (testReport.md)
### テスト実行日時
YYYY-MM-DD HH:MM:SS

### テスト結果サマリ
- 総テストケース数: XX
- 成功: XX
- 失敗: 0
- スキップ: XX

### カバレッジ
- ライン カバレッジ: XX%
- ブランチ カバレッジ: XX%
- 目標達成度: ✅ / ⚠️

### 実施したテストケース
| ID | 分類 | 結果 | 備考 |
|----|------|------|------|
| TC01 | 正常系 | ✅ | - |

### 品質評価
- [ ] すべてのテストケースが成功
- [ ] カバレッジ目標を達成
- [ ] 異常系・境界値テストを実施
- [ ] パフォーマンス要件を満たす

### コミット推奨度
- [ ] **コミット可** (品質基準を満たしている)
- [ ] **要追加テスト** (不足している観点がある)
