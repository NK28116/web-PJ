# 要件

## 機能要件

### 1. 「投稿を編集」機能
- 場所:`components/templates/PostTemplate/PostDetailModal.tsx`
- 「投稿を編集」ボタンを押す
  - `post.content`を編集できる
  - `post.tags`を編集できる
  - `画像エリア`の画像を変更できる
  - 「投稿を編集」ボタンが「編集内容を保存する」になる
  - 「非表示にする」ボタンが「編集を一時保存する」になる
  - `onClose`ボタンが「投稿を削除する」になる
  - モーダル外部をタップすると「はい」「いいえ」を持つ「編集をキャンセルしますか？」のモーダルウインドウが新た画面中央に表示される
